# Librarymanagement.java
PROJECT NAME
Library Management System – Java

------------------------------------------------------------

INTRODUCTION

Library Management System is a desktop application developed in Java.  
It helps in managing library records such as:

• Adding books  
• Searching books  
• Managing members  
• Issue and return records  

The system provides an easy and efficient way to handle library data.

------------------------------------------------------------

FEATURES

✔ Add new books  
✔ View all books  
✔ Search book by ID or name  
✔ Update book information  
✔ Delete book record  
✔ User-friendly interface  
✔ File based data storage  

------------------------------------------------------------

TECHNOLOGIES USED

• Java Programming Language  
• Swing / AWT (if GUI)  
• File Handling  
• Object Oriented Programming  

------------------------------------------------------------

MODULES

1. Add Record Module  
2. View Record Module  
3. Search Module  
4. Update Module  
5. Delete Module  
6. File Storage Module  

------------------------------------------------------------

HOW TO RUN

1. Compile the program:

   javac Librarymanagement.java

2. Run the program:

   java Librarymanagement

------------------------------------------------------------

HOW TO USE

1. Run the program  
2. Select option from menu  
3. Enter required data  
4. System will store data in file  
5. You can search/update/delete anytime  

------------------------------------------------------------

ADVANTAGES

• Easy to use  
• Fast processing  
• Permanent data storage  
• Reduces manual work  

------------------------------------------------------------

LIMITATIONS

• Works on text file database  
• Single user system  
• No online connectivity  

------------------------------------------------------------
